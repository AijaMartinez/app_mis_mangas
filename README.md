# 📚 Manga Explorer App

Aplicación web que consume una **API pública de mangas** para mostrar información sobre diferentes títulos.  
La aplicación permite a los usuarios **registrarse o iniciar sesión**, buscar mangas, aplicar filtros y guardar sus favoritos.

---

## 🚀 Características

- 🔐 **Autenticación de usuarios**
  - Registro
  - Login
  - Sesión persistente

- 📖 **Exploración de mangas**
  - Consumo de una **API pública de mangas**
  - Visualización de títulos, portada e información básica

- 🔎 **Búsqueda**
  - Buscar mangas por nombre

- 🎛 **Filtros**
  - Filtrar mangas por diferentes criterios (ej: género, estado, popularidad, etc.)

- ⭐ **Gestión de favoritos**
  - Guardar mangas en tu lista personal
  - Eliminar mangas guardados

---

## 🧱 Arquitectura de la aplicación

La aplicación sigue una arquitectura basada en **cliente-servidor**.
